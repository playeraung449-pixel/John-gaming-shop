အသုံးပြုနည်း

၁
အကောင့်ဖွင့်ခြင်း
Signup page တွင် အမည်၊ ဖုန်း၊ အီးမေးလ် တို့ကို ဖြည့်သွင်းပြီး အကောင့်သစ် ဖွင့်လှစ်နိုင်ပါသည်။
၂
ငွေဖြည့်သွင်းခြင်း
KBZ Pay သို့မဟုတ် Wave Pay ဖြင့် ငွေလွှဲပြီး ပြေစာ (Slip) ပုံကို တင်ပေးရပါမည်။ Admin မှ စစ်ဆေးပြီး ၅ မိနစ်အတွင်း ငွေဖြည့်ပေးပါမည်။
၃
ဂိမ်းပစ္စည်းဝယ်ယူခြင်း
နှစ်သက်ရာဂိမ်းကို ရွေးချယ်ပြီး Game ID နှင့် Server ID တို့ကို မှန်ကန်စွာ ဖြည့်သွင်းကာ ဝယ်ယူနိုင်ပါသည်။
၄
Spin Wheel
Spin Token များရှိပါက Spin Wheel လှည့်ပြီး ဆုလာဘ်များ ကံစမ်းနိုင်ပါသည်။
၅
မှတ်တမ်းကြည့်ခြင်း
မိမိဝယ်ယူခဲ့သော မှတ်တမ်းများကို 'မှတ်တမ်း' ကဏ္ဍတွင် အသေးစိတ် ပြန်လည်ကြည့်ရှုနိုင်ပါသည်။
၆
စကားဝှက်ပြောင်းခြင်း
မိမိအကောင့် လုံခြုံရေးအတွက် စကားဝှက်ကို အချိန်မရွေး ပြောင်းလဲနိုင်ပါသည်။
